Website Planning Document

Project Subject: Guayaquil’s Hidden Gems

Site Name

Selected Name: Guayaquil’s Hidden Gems: A Local Travel Guide

Reason for Selection:

I chose this topic because I live in Guayaquil and I really enjoy finding places that aren't just the typical tourist spots. A list of locations is perfect for practicing what we are learning in class, like JavaScript arrays, objects, and event listeners for the form. It lets me meet all the project requirements while building something about my own city.

Site Purpose

The purpose of this 3-page website is to help visitors and locals find cool, lesser-known spots in Guayaquil, like small coffee shops, nice views, and quiet parks. The site delivers value by dynamically loading content, filtering by categories, and allowing user interaction through suggestions.

Planned Page Architecture:

  • Page 1 (Home): A simple welcome page introducing the concept, with a small JavaScript feature that changes the greeting depending on the time of day.
  • Page 2 (The Gems Directory): A page showing all the places. I will use a JavaScript array of objects to dynamically generate the place cards using template literals. I'll also add buttons so users can filter the spots by category (like Cafes or Parks), and use lazy loading for the images.
  • Page 3 (Suggest a Spot): A page with an HTML form where users can submit their own favorite places. I will use JavaScript to validate the input and use localStorage to save and show how many times a user has submitted a spot.

Target Audience Scenarios

This website planning addresses specific questions from our target audience:

  • Scenario Question 1: "Where can I find a quiet park or a small, non-commercial coffee shop in Guayaquil away from typical crowded tourist hubs?"
  • Scenario Question 2: "How can I contribute to this guide and suggest my own favorite local hidden spot using an online submission form?"

Color Scheme

The site uses colors inspired by Guayaquil's tropical environment and local identity, ensuring high contrast and clean presentation:

  • Primary Color: #0056b3 (Guayaquil Sky Blue) — Used for main navigation layouts and header sections.
  • Secondary Color: #20c997 (Tropical Teal) — Used for section subheadings, filter highlights, and active borders.
  • Accent Color: #ffc107 (Sun Gold) — Used for important buttons and dynamic alerts.
  • Text Color: #212529 (Dark Charcoal) — Applied to all primary body content for structural contrast and readability.

Typography

To keep the site fast and clean across modern screens, we utilize a system sans-serif font family layout:

  • Heading Font Selection: "Segoe UI", Roboto, sans-serif — Applied to structural titles (H1, H2) for clean line visual weight.
  • Body Font Selection: "Segoe UI", Helvetica, sans-serif — Applied to standard paragraphs, interactive menus, and text fields for digital legibility.
